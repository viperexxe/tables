# LeMage Tables - Premium phpBB Extension

Premium BBCode Tables for phpBB with responsive design, glassmorphism aesthetics, and easy-to-use editor integration.

## Features
- **Responsive Design**: Tables adapt perfectly to mobile, tablet, and desktop views.
- **Glassmorphism**: Modern UI with subtle blurs and semi-transparent backgrounds.
- **Editor Integration**: Adds a "Tabelle" button to the posting editor for quick template insertion.
- **Custom BBCodes**: Supports [table], [tr], [th], and [td].

## Installation
1. Download the repository.
2. Copy the `lemage` folder into your phpBB `ext/` directory.
3. The resulting path should be `ext/lemage/tables/`.
4. Navigate to the **ACP -> Customise -> Manage extensions**.
5. Find **LeMage Tables** and click **Enable**.
